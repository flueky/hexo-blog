## hexo-blog

hexo 博客示例

第一次使用，需要执行 `npm install` 安装 **hexo** 有关文件。

### 版本说明

#### v0.0.1

初始 hexo 配置。

#### v0.0.2

使用 material-x 主题。

主题直接拷贝自 `xaoxuu`。

```Shell
git clone https://github.com/xaoxuu/hexo-theme-material-x themes/material-x
```

主题使用了搜索功能，需要安装依赖。

```Shell
npm i -S hexo-generator-search hexo-generator-json-content
```
